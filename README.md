# 🧮 Calculator Web App

This project is a **responsive calculator** built as part of my **Web Development Practice / Internship Task** using pure HTML, CSS, and JavaScript.

---

## 📌 Project Overview

This is a simple yet functional calculator that performs basic arithmetic operations.
The project focuses on **DOM manipulation, event handling, and responsive UI design**.

---

## ✨ Features

* Basic operations: Addition, Subtraction, Multiplication, Division
* Clear (C) and Delete (⌫) functionality
* Responsive design (works on mobile & desktop)
* Clean and minimal UI
* Error handling for invalid inputs

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

---

```
project/

└── index.html   (Contains HTML, CSS & JS)
```
